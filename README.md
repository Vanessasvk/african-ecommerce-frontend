1. Project Overview
This project is a full-scale e-commerce platform dedicated to selling African-related products. The design was finalized in Figma, focusing on a cultural aesthetic that highlights "New Arrivals," "Sales," and specific African categories like clothing and jewelry.

2. Frontend Distribution (22 Pages)
We have split the workload into three specialized roles to ensure consistency and efficiency:

Developer 1 (Blue): The Transaction & Setup Lead

Focus: Landing screens, Authentication (Login/Sign-up), and the Checkout/Payment flow.

Developer 2 (Purple): The Brand & Discovery Lead

Focus: Home Page content, Testimonials ("Happy Customers"), and Informational pages like Contact/About.

Developer 3 (Green): The Product & Catalog Architect

Focus: Product Listings, Detailed Product views (e.g., Buba Blouse), and User Dashboards.
